# Characters

## Name-type system (chapters 1–3 scope)

This book uses three classes of personal-name rendering. Every entry below is classified into exactly one class, and the classification drives the English form. Walk-on characters and chapters past 3 are out of scope for this system; do not extrapolate.

1. **Full social/legal personal name** (family name + given name in the Chinese source). For chapters 1–3, render in **Western order: Given name + Family name**, e.g. 秦流西 → "Liuxi Qin". The original source-order form (e.g. "Qin Liuxi") is treated as a deprecated variant and registered in the entry's Notes line so consistency audit can catch drift.
2. **Given-name-only attendant / servant / side character** (single Chinese given name with no family-name pairing in the source). Preserve source order, **spaced by Chinese character**, e.g. 岐黄 → "Qi Huang", 陈皮 → "Chen Pi". Collapsed one-word forms (e.g. "Qihuang") are deprecated variants.
3. **Daoist / religious / courtesy / title-bearing name** (老道 / 道人 / 大师 / 子-type names and surname-plus-functional-title forms such as 张大夫 → "Doctor Zhang"). Preserve source order; render the title or sobriquet element naturally in English. These are not migrated to Western order.

Family-status forms (姓 + 氏 / 姓 + 老太太, e.g. 王氏 → "Lady Wang", 秦老太太 → "Old Lady Qin") are not full given+family names and are out of class 1; they keep their existing forms.

## Confirmed / working renderings

### Liuxi Qin
- Chinese: 秦流西
- English rendering: Liuxi Qin
- Name type: full social/legal personal name (class 1) — Western order per ch 1–3 system.
- Notes: Main character. Naming-system migration: ch 1–3 Western order, canonical is Liuxi Qin. Do not drift to "Qin Liuxi" (deprecated source-order form, superseded by Western-order migration) or "Qi Liuxi" or other variants.

**Address forms for 秦流西:**

| Context | Chinese form | English rendering |
|---|---|---|
| Full name (narrative / formal) | 秦流西 | Liuxi Qin |
| Familiar address (by elders, affection) | 西丫头, 丫头, 你这丫头 | Xi'er |
| Generic noun reference (narrative, 3rd-party observer) | 这个姑娘 / 她 | this young woman / she (contextual; prefers neutral form unless register demands closer familiarity) |

- Use "Liuxi Qin" as the narrative default. Reserve "Xi'er" for familiar direct address (elder speaking to her). Do not over-use "Xi'er" in narrative voice or from characters who would not use familiar address.
- Generic references (这个姑娘, 那姑娘, 她 as "the young woman") are deliberately neutral — do not replace them with "Xi'er" or "Liuxi Qin" unless the context clearly calls for named reference. The generic form is a valid register choice in the source.
