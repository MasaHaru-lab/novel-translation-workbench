# Titles and Terms

## Working system

### 大小姐
- Chinese: 大小姐
- Working English rendering: Young Lady
- Notes: Keep as “Young Lady” for now. Do not map this mechanically onto British aristocratic forms such as “Lady + given name.” This is a household-status term, not a British courtesy title. In running prose, wording may vary slightly if rank order or family structure needs to be made clearer.

### 嫡母
- Chinese: 嫡母
- Working English rendering: legal mother
- Notes: Rejected alternatives: “principal mother” (too legalistic/unnatural), plain “mother” (loses hierarchy). “legal mother” preserves the legal-wife / principal-wife distinction from concubine-born structure without sounding legalistic. Do not use “principal mother”.

### 长房
- Chinese: 长房
- Working English rendering: main branch
- Notes: Keep stable unless a better clan-system rendering is adopted consistently.

### 嬷嬷
- Chinese: 嬷嬷
- Working English rendering: Momo
- Notes: For now, prefer retaining “Momo” in role names such as “Ding Momo” rather than forcing “Nanny” or “Nurse.” Final treatment should stay aligned with the broader system for household titles and forms of address.

### 光禄寺卿
- Chinese: 光禄寺卿
- Working English rendering: Chief Minister of the Court of Imperial Entertainments
- Notes: Provisional. Official title system needs later review for consistency and readability.

### 贵妃
- Chinese: 贵妃
- Working English rendering: Noble Consort
- Notes: Prefer stable court-rank terminology once adopted.

### 丫头 / 西丫头 / 你这丫头
- Chinese: 丫头 / 西丫头 / 你这丫头
- Working English rendering: Xi’er
- Notes: Canonical rendering is "Xi’er" — preserves the elder-to-younger familiar register while reading naturally in English dialogue. The character's given name serves as the consistent address form. Rejected alternatives: "girl" (too flat, fails register), "little girl" (infantilizing), "Little Xi" (over-Westernized). Do not use any of these rejected forms.

### 清平观
- Chinese: 清平观
- Working English rendering: Qingping Temple
- Notes: Canonical rendering is "Qingping Temple" (not "Qingping Abbey"). Qin Liuxi's temple. Keep stable across all chapters — do not switch between Temple, Abbey, Monastery, or Shrine. "Qingping Abbey" appeared in earlier draft output and must be corrected.

### 老二家的
- Chinese: 老二家的
- Working English rendering: Sentence-level recast — use character's name or natural address (see style notes §"Chinese household-address terms")
- Notes: Household address term referring to the wife of the second son/brother. Literal renderings like "Second Branch" produce unnatural English fiction. Recast the sentence using the character's name (Lady Xie) or a natural English address in dialogue — the address label itself does not survive translation. Discovered in Stage 5 real-model validation (DeepSeek V4 Flash, 2026-05-02): hard quality gates (pipeline, quality gate, consistency check) all passed, but literary review flagged the gap. Accepted fix: Old Lady Qin addresses Lady Xie directly — "You take Concubine Pan and the others to settle the children first."

### 大夫 (physician title)
- Chinese: 大夫
- Pattern (instantiate, do not emit verbatim): Doctor + surname (e.g. Doctor Zhang, Doctor Gao, Doctor Yang)
- Notes: Render the physician title as "Doctor" followed by the actual surname — Doctor Zhang, Doctor Gao, Doctor Yang. Never "Dr." (abbreviation), never "Zhang Dafu" (transliteration), and never the literal pattern with brackets. The existing project convention for household roles uses functional English titles (Lady Wang, Ding Momo, Aunt Li); matching "Doctor + surname" keeps the system consistent. This is a working choice — see unresolved_decisions for the ongoing discussion of the 大夫 address system.
